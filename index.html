Eis a minha primeira página Web!

A cadeira de Desenvolvimento de Interfaces Web é essencial no front-end, pois ensina a criar páginas estruturadas com HTML, estilizadas com CSS e interativas com JavaScript. Sua importância está em garantir experiências acessíveis, responsivas e agradáveis para o usuário. A genialidade desse campo está em transformar código em interfaces intuitivas, unindo lógica e criatividade.

Na Unidade Curricular de Desenvolvimento de Interfaces Web abordaremos as seguintes tecnologias Web 🌐:

  • HTML
  • CSS
  • JavaScript
  • React e Next.js

Iremos criar soluções criativas!
